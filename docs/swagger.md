# Interactive API Reference

The interactive API reference is powered by **Swagger UI** and rendered directly from the OpenAPI 3.0 specification.

Use it to:

*   Browse all available endpoints
*   View request and response schemas
*   Try out live API calls directly from your browser (add your API key in the Authorize dialog)

## Open the Interactive Reference

**[Open Swagger UI](swagger-ui/index.html)**

## Using Swagger UI

1.  Click **Authorize** in the top right and enter your `X-API-Key`.
1.  Select an endpoint to expand it.
1.  Click **Try it out**, fill in any parameters, and click **Execute**.
1.  View the response body, status code, and headers.

## Download the OpenAPI Spec

The raw OpenAPI 3.0 specification is available at:

```text
openapi/v1/openapi.yaml
```

You can import this file into Postman or any OpenAPI-compatible tooling.
